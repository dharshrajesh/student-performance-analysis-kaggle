
# Student Performance Analysis using Python
## Overview
This project analyzes student performance data to understand how factors such as study habits, attendance, family background, and lifestyle influence final grades. Using Python and exploratory data analysis (EDA), the dataset is explored to identify meaningful trends and patterns.

## Dataset

* **Records:** 649 students
* **Columns:** 33
* **Target Variable:** G3 (Final Grade)

## Tools & Libraries

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

## Project Workflow

* Loaded and explored the dataset
* Checked for missing values and duplicates
* Performed statistical analysis
* Created visualizations to identify trends
* Summarized key insights

## Key Insights

* Previous grades (G1 and G2) had the strongest impact on the final grade.
* Students with previous failures generally scored lower.
* Study time showed a positive relationship with academic performance.
* Higher absences were often associated with lower final grades.
